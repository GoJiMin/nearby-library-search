- Phase 4-2는 코드 구현이 아니라 MVP UX/UI 설계를 결정 완료 상태로 고정하는 단계다.
- `.impeccable.md`와 `AGENTS.md`의 Design Context를 설계 source of truth로 사용한다.
- Phase 5는 Phase 4-2 문서를 기준으로 구현한다.

- 문서만 읽고 각 상태의 표현과 다음 행동을 구분할 수 있어야 한다.

## Assumptions
