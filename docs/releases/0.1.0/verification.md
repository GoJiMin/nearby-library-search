# 0.1.0 Verification

이 문서는 `0.1.0`을 배포 가능한 상태로 판단할 때 사용한 근거를 정리한 기록입니다. 목적은 “대충 확인했다”를 적는 것이 아니라, 무엇을 다시 실행했고 그 결과가 어땠는지를 다음 개발자가 그대로 이해할 수 있게 남기는 데 있습니다.

## 배포 가능하다고 판단한 이유

`0.1.0`은 아래 세 가지를 통과했기 때문에 배포 가능한 상태로 판단했습니다.

- 프로젝트 내 전체 타입 점검과 웹과 서버가 정상적으로 빌드됩니다.
- 웹 코드베이스 전체 테스트와, 그 안에 포함된 핵심 통합테스트가 통과했습니다.
- 서버 코드베이스 전체 테스트와, 그 안에 포함된 핵심 라우트 통합테스트가 통과했습니다.

즉 `0.1.0`은 화면만 만들어 둔 상태가 아니라, 검색부터 대출 가능 여부 확인까지 필요한 기능이 타입 점검, 빌드, 전체 테스트를 통과한 상태입니다.

## 무엇을 검증했는가

### 1. 타입 점검과 빌드

먼저 아래 세 가지를 다시 실행했습니다.

- `pnpm exec tsc -b tsconfig.json`
- `pnpm --filter @nearby-library-search/web build`
- `pnpm --filter @nearby-library-search/bff build`

이 확인으로 아래를 판단할 수 있습니다.

- web, bff, contracts가 같은 타입 정의를 기준으로 컴파일되는지
- 지금 저장소 상태로 web 배포 산출물이 다시 만들어지는지
- 지금 저장소 상태로 bff 배포 산출물이 다시 만들어지는지

결과는 세 항목 모두 통과였습니다. 즉 지금 저장소 상태로는 타입 단계와 배포 산출물 생성 단계에서 막히지 않습니다.

### 2. web 전체 테스트

web은 일부 테스트만 따로 고르지 않고, 현재 코드베이스 전체 테스트를 다시 실행했습니다. 이 실행에는 검색 시작, 검색 결과, 재검색, 페이지 이동, 상세 보기, 지역 선택, 도서관 결과, 대출 가능 여부 확인을 검증하는 통합테스트가 함께 포함되어 있습니다.

실행에는 아래 명령어를 사용했습니다.

```bash
pnpm --filter @nearby-library-search/web test:coverage
```

전체 실행 결과는 아래와 같습니다.

- 테스트 파일 28개 통과
- 테스트 234개 통과

즉 web 쪽에서는 검색 시작부터 결과 확인, 상세 보기, 지역 선택, 도서관 결과, 대출 가능 여부 확인까지 현재 문서에서 설명한 주요 기능을 전체 테스트로 다시 확인했습니다.

### 3. bff 전체 테스트

bff도 일부 라우트만 고르지 않고, 현재 코드베이스 전체 테스트를 다시 실행했습니다. 이 실행에는 책 검색, 책 상세, 도서관 검색, 대출 가능 여부 조회, 외부 도서 Open API 이상 응답 처리를 검증하는 통합테스트가 포함되어 있습니다.

실행에는 아래 명령어를 사용했습니다.

```bash
pnpm --filter @nearby-library-search/bff test:coverage
```

전체 실행 결과는 아래와 같습니다.

- 테스트 파일 22개 통과
- 테스트 114개 통과

즉 bff 쪽에서는 정상 응답뿐 아니라 외부 도서 Open API가 비어 있거나 잘못된 값을 줄 때도, 현재 서비스가 기대하는 오류 메시지와 응답으로 다시 정리되는지까지 전체 테스트로 확인했습니다.

## 테스트 커버리지

| App | Statements | Branches | Functions | Lines | Average |
| --- | ---: | ---: | ---: | ---: | ---: |
| Web | 91.65% | 85.82% | 90.11% | 91.39% | 89.74% |
| BFF | 92.44% | 83.94% | 99.20% | 92.34% | 91.98% |
| Overall | 92.05% | 84.88% | 94.66% | 91.87% | 90.86% |

- 이 수치는 `0.1.0`을 통과/실패로 가르는 기준이 아니라, 이후 버전에서 비교할 기준선입니다.

## 최종 판단

`0.1.0`은 지금 기준에서 배포 가능한 상태입니다. 판단 근거는 아래와 같습니다.

- 타입 점검과 배포 산출물 생성이 다시 통과했습니다.
- 현재 구현된 web 코드베이스 전체 테스트가 통과했습니다.
- 현재 구현된 bff 코드베이스 전체 테스트가 통과했습니다.

즉 `0.1.0`은 **검색부터 대출 가능 여부 확인이라는 핵심 기능이 동작함이 검증된 MVP**입니다.
